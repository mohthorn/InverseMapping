usage: ./pr01 'filename' 'method' 'parameter1' 'parameter2'

rotate only uses one parameter, the scale should be 0-360

mirror doesn't use parameters

usable commands examples:
./pr01 02.PPm scale 1.5 0.7
./pr01 02.PPm rotate 45 0
./pr01 02.PPm shear 0.5 0.5
./pr01 02.PPm mirror 0 0
./pr01 02.PPm translation 400 400
./pr01 02.PPm perspective 1 1
